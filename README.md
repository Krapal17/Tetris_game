# 🎮 Tetris Game (Python)

A simple and fun **Tetris Game** built using Python and pygame.

---

## 🚀 Features

* Classic Tetris gameplay
* Smooth block movement
* Rotation system
* Line clearing
* Simple and clean UI

---

## 🛠️ Technologies Used

* Python 🐍
* pygame 🎮

---

## 🎮 Controls

* ⬅️ Left Arrow → Move Left
* ➡️ Right Arrow → Move Right
* ⬇️ Down Arrow → Fast Drop
* ⬆️ Up Arrow → Rotate
