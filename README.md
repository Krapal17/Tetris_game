# 🎮 Tetris Game (Python)

A simple and fun **Tetris Game** built using Python and pygame.

---

## 🚀 Features

* Classic Tetris gameplay
* Smooth block movement
* Rotation system
* Line clearing
* Simple and clean UI

---

## 🛠️ Technologies Used

* Python 🐍
* pygame 🎮

---

## 📦 Installation

1. Clone the repository:

```
git clone https://github.com/Krapal17/Tetris_game.git
```

2. Navigate to the folder:

```
cd Tetris_game
```

3. Install dependencies:

```
pip install pygame
```

---

## ▶️ How to Run

```
python tetris.py
```

---

## 🎮 Controls

* ⬅️ Left Arrow → Move Left
* ➡️ Right Arrow → Move Right
* ⬇️ Down Arrow → Fast Drop
* ⬆️ Up Arrow → Rotate
